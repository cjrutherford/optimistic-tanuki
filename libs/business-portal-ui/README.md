# business-portal-ui

`business-portal-ui` is a shared Angular UI library for business owner and client portal flows, including dashboard, client management, availability, billing, and site-editor pages.

## Public API

The library exposes owner and client portal page components plus the portal shell used by host apps such as `business-site`.
