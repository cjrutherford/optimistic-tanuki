# Social UI

`social-ui` contains reusable social components, models, services, types, and styles. Its source lives under `libs/social-ui/src/lib`.

## Repo Role

- shared frontend support for social features
- companion library to the `social` backend service and gateway social flows

## Key Surfaces

- feed, comment, reaction, and social interaction components
- shared models and frontend support code for social workflows

## Documentation

- generated API reference in `ui-playground`: `/docs/api/social-ui`

## Nx Commands

```bash
pnpm exec nx build social-ui
pnpm exec nx test social-ui
```
