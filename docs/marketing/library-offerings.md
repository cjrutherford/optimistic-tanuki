# Library Offerings

Optimistic Tanuki library surfaces are intended to be provided free through npm where they are approved for publication.

## Distribution Posture

- free npm packages
- source development in the main monorepo
- public release publication from the dedicated mirror repository

## Current Examples

### Billing SDK

- package: `@optimistic-tanuki/billing-sdk`
- role: typed client helpers and usage workflow primitives for billing integrations
- pricing posture: free npm package
- release posture: mirrored out of the monorepo for public distribution
