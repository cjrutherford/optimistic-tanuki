# Pricing Models

This project currently documents pricing posture for hosted service surfaces without implementing commercial enforcement.

## Hosted Service Pricing Shapes

- metered usage: pay for measured events, requests, or units consumed
- block usage subscription: recurring plan that grants a fixed quota block each billing period
- unlimited subscription: flat recurring price for bounded fair-use hosted access

## Self-Hosted Positioning

Hosted plans can be offered for managed operations while self-hosted Docker containers remain available for teams that want to run the service themselves.

## Current Usage

- `billing-service`: metered hosted posture with a visible self-hosted Docker option
- future hosted services can adopt the same vocabulary when they are framed as managed product surfaces
