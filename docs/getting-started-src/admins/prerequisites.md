# Prerequisites

Install the following before bootstrapping the stack:

- Docker Desktop or Docker Engine with Compose support
- Node.js 18 or newer
- `pnpm`
- Git

Verify the versions you expect:

```bash
docker --version
node --version
pnpm --version
git --version
```
