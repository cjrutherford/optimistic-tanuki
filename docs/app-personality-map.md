# App Personality Map

This matrix records recommended default personalities for Angular client apps. It uses the personality definitions in `libs/theme-lib/docs/PERSONALITY_SYSTEM.md`.

| App                       | Default personality | Rationale                                                                                |
| ------------------------- | ------------------- | ---------------------------------------------------------------------------------------- |
| client-interface          | `soft-touch`        | Community/social surfaces benefit from warm, approachable visuals.                       |
| owner-console             | `control-center`    | Operator dashboards need technical density, grid structure, and clear controls.          |
| system-configurator       | `control-center`    | Hardware configuration is technical and benefits from precise dashboard affordances.     |
| local-hub                 | `soft-touch`        | Local community/commerce should feel warm, organic, and approachable.                    |
| developer-portal          | `foundation`        | Documentation and API onboarding should prioritize clarity and neutral layout.           |
| christopherrutherford-net | `elegant`           | Personal/editorial consulting content benefits from refined typography and premium tone. |
| business-site             | `professional`      | B2B public and portal flows need trustworthy enterprise defaults.                        |
| forgeofwill               | `bold`              | Branded productivity experience needs energetic contrast and strong accents.             |
| digital-homestead         | `soft-touch`        | Homesteading/community content aligns with organic warmth and pill-shaped softness.      |
| hai                       | `foundation`        | Owned-computing messaging should remain clear, practical, and minimal.                   |
| marketing-generator       | `control-center`    | Generator/editor workflows benefit from technical dashboard density.                     |
| fin-commander             | `professional`      | Finance workflows need conservative, trustworthy defaults.                               |
| leads-app                 | `control-center`    | Lead discovery and analytics are command-center workflows.                               |
| business-configurator     | `professional`      | Business setup should feel stable, guided, and enterprise-ready.                         |
| configurable-client       | `foundation`        | Tenant shells need a neutral baseline that can accept tenant branding.                   |
| store-client              | `playful`           | Commerce/customer surfaces benefit from friendly energy and approachable interactions.   |
| video-client              | `electric`          | Video discovery and creator surfaces need kinetic, vibrant personality.                  |
| d6                        | `soft-touch`        | Reflection/wellness practice needs calm, gentle visuals.                                 |
