# Optimistic Tanuki Public Packages

This repository is the public mirror for npm-ready Optimistic Tanuki packages.

It is generated from the internal development monorepo and only contains approved package sources, release metadata, and package release automation.

## Available Commands

```bash
pnpm install --no-frozen-lockfile
pnpm test
pnpm build
pnpm version
pnpm publish
```

## Release Ownership

Versioning and npm publication happen from this mirror repository. Source development continues in the internal monorepo.
